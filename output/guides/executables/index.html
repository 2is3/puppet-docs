Executables

Puppet ships with a number of useful executables.

Contents

  1. Daemons
  2. Utilities
  3. Documentation

Daemons

  • puppetmasterd: Puppet master daemon
  • puppetd: Puppet client daemon

Utilities

  • ralsh: Use the Puppet resource abstraction layer to directly interact with the system.
  • filebucket: A stand-alone filebucket client for sending files to a local or central filebucket.

Documentation

  • pi: Print help about puppet types on the console.
  • puppetdoc: Generate in-depth puppet documentation (also see the online docs)
