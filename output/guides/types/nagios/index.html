Nagios Types

  • These types generate Nagios configuration statements in Nagios-parseable configuration files.
  • Autogenerated using the model developed in Naginator.

Contents

  1. Important Notes
  2. Types

Important Notes

You can purge Nagios resources using the resources type, but only in the default file locations. This is an architectural limitation.
